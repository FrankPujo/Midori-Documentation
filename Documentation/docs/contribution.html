Contribute to Midori Docs

Hi! Thank you for considering contributing to the Midori Documentation!
This project aims to create a complete Documentation for the Midori Browser
It is also completely community-driven, so your contribution are very important.

How can I contribute?

To contribute you first have to visit our GitHub Repository.
Then you can follow two methods:

  1. Fork our repository, add an html file for a new topic to add or modify pre-existant html's and then pull a request on our repository (suggested for bigger changes).
  2. Create an issue on our repository to advice a new topic to add or signal a mistake.(suggested for smaller changes).

In the repository README.md you will find a complete guide to the contribution but here we sum up the content:
always follow the template.html file you find in the repository to create a new .html and the docs.css for styles as well.
